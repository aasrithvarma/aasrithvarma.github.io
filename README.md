# Dr. Tanmai Aasrith Varma Ayenampudi - Academic Digital Identity

Production-ready static architecture for a 30+ year academic medical career. Built exclusively with HTML5, CSS3, ES Modules, and Node.js for zero-dependency longevity.

## Local Development
1. `npm install`
2. `npm run dev` (Builds the static files and serves them locally)

## Deployment
Pushing to the `main` branch automatically triggers the GitHub Actions workflow to build the Markdown and deploy to GitHub Pages.
